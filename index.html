CSS Transition

CSS Transition memungkinkan perubahan nilai properti CSS terjadi dengan halus selama durasi yang ditentukan.

Untuk membuat efek transisi, Kita harus menentukan dua hal:

  • Properti CSS yang ingin di tambahkan efek
  • Durasi dari efeknya

Dalam kasus diatas kita nge transisiin 'transform' dan 'background' dengan waktu 300ms/0.3s

Kita bisa aja pakai 'all' untuk mempersingkat


Yang ini kita pakai 'all'

.transition-box {
    width: 100px;
    height: 100px;
    background: #6366f1;
    transition: transform 0.3s ease, background 0.3s ease;
}

.transition-box:hover {
    transform: scale(1.2) rotate(45deg);
    background: #4f46e5;
}
  
transition: all 0.3s ease;

Property dari Transition

Property Value Deskripsi
transition-property none | all | property-name Menentukan properti CSS yang akan ditransisiin
transition-duration time (bisa s atau ms, bebas) Durasi transisi
transition-timing-function ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier() Kurva kecepatan transisi
transition-delay time (bisa s atau ms, bebas) Waktu transisinya di tunda

Yang kita pakai di Contoh kode adalah Shorthand dari semua property di atas

CSS Animation

CSS Animation memungkinkan elemen dianimasi tanpa menggunakan JavaScript atau trigger seperti hover, dan animation bisa kita bikin ngulang terus.

Bila kita menentukan style CSS di dalam aturan @keyframes, animasi akan berubah secara bertahap dari gaya saat ini ke gaya baru pada waktu tertentu. Agar animasi berfungsi, kita harus mengikat animasi ke suatu elemen. 

.animation-box {
    width: 100px;
    height: 100px;
    background: #8b5cf6;
    animation: bounce 2s infinite;
}

@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-30px);
    }
}

Property Animation

Property Value Deskripsi
animation-name keyframename | none Nama @keyframes yang digunakan
animation-duration time (bisa s atau ms, bebas) Durasi satu siklus animasi
animation-timing-function ease | linear | ease-in | ease-out | ease-in-out | steps() Kurva kecepatan animasi
animation-delay time (bisa s atau ms, bebas) Waktu tunda sebelum animasi dimulai
animation-iteration-count number | infinite Berapa kali animasi diulang
animation-direction normal | reverse | alternate | alternate-reverse Arah pemutaran animasi
animation-fill-mode none | forwards | backwards | both Bagaimana animasi menerapkan gaya sebelum dan sesudah eksekusi
animation-play-state running | paused Status pemutaran animasi

Yang kita pakai di Contoh kode adalah Shorthand dari semua property di atas

CSS Keyframe

@keyframes digunakan untuk mendefinisikan tahapan dalam sebuah animasi CSS, nah keyframe gak bisa kalian pakai tanpa sebuah property 'animation' jadi mereka tuh dah pasangan.


'0%-100%' itu namanya keyframe-selector, disini kalian bebas mau kasih berapa keyframe-selectornya. Kalian juga bisa pake 'from' dan 'to' itu sama aja kayak '0%' dan '100%' 

.keyframe-box {
    width: 100px;
    height: 100px;
    background: #ec4899;
    animation: colorChange 4s infinite;
}

@keyframes colorChange {
    0% {
        background: #ec4899;
        transform: scale(1);
    }
    25% {
        background: #6366f1;
        transform: scale(1.2);
    }
    50% {
        background: #8b5cf6;
        transform: scale(1);
    }
    75% {
        background: #14b8a6;
        transform: scale(1.2);
    }
    100% {
        background: #ec4899;
        transform: scale(1);
    }
}

Property Experimental

Property Status Deskripsi
animatin-timeline Experimental Properti CSS animation-timeline menentukan garis waktu yang digunakan untuk mengontrol kemajuan animasi CSS.
animatin-range Experimental Mengatur awal dan akhir rentang lampiran animasi sepanjang garis waktunya

Gua gak nyaranin kalian pake fitur experimental, karena fitur experimental biasanya fitur yang dikembangin browser tertenu jadi cuman beberapa browser yg support. Tunggu aja sampai di kembangin sama yang lain
